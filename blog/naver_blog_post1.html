🔮 사주팔자란?

사주팔자(四柱八字)는 태어난 년·월·일·시의 네 기둥(四柱)과 여덟 글자(八字)로 한 사람의 운명과 성격, 적성을 분석하는 동양 전통 학문입니다.

저희 서비스는 정밀한 만세력 알고리즘으로 사주를 분석하여 누구나 쉽게 자신의 사주를 이해할 수 있도록 만들었습니다.

✨ 주요 분석 기능 21가지

📅

오늘의 운세 달력

월간 달력에 매일 운세 점수를 색상으로 표시. 날짜 클릭 시 시간대별 상세 운세, 행운의 색상·방향·숫자까지.

⚖️

용신 분석

신강/신약 판별 후 용신·희신·기신을 정밀 선정. 행운의 방향, 색상, 숫자, 계절, 음식, 직업군까지 추천.

💪

일간 강약 분석

득령·득지·득세·통근 4가지 기준으로 일간의 강약을 점수화. 극강~극약 5단계 판별.

📊

년운 · 월운 · 일운

년운 5대 카테고리(재물·직장·연애·건강·학업) + 월 클릭 시 달력 펼침 + 일별 시간대 운세.

🛤️

인생 로드맵

초년·청년·중년·말년 4단계 운세 + 대운 기반 인생 전환점(전성기·새출발·재물운) 분석.

⭐

신살 30종 분석

천을귀인·도화살·역마살·괴강살 등 30종 이상의 귀인성과 신살을 분류하여 상세 해석.

💑

궁합 분석

두 사람의 사주를 비교하여 오행 상생상극, 육합·충, 오행 보완도를 종합 분석합니다.

🕐

진태양시 보정

서울 기준 경도 보정 + 균시차를 적용하여 더 정밀한 시주를 계산합니다.

📋 전체 분석 항목

✅ 사주 원국 (四柱原局) ✅ 오늘의 운세 달력 ✅ 일간 강약 분석 ✅ 용신 분석 (用神) ✅ 합충형파해 ✅ 격국 (格局) ✅ 60일주론 ✅ 오행 분석 (차트) ✅ 납음오행 ✅ 공망 (空亡) ✅ 십성 분석 ✅ 십이운성 ✅ 대운 (大運) ✅ 월운 (달력 연동) ✅ 년운 (5대 카테고리) ✅ 토정비결 ✅ 세운 10년 흐름 ✅ 인생 로드맵 ✅ 신살 30종 ✅ 종합 해석

📖 사용 방법

1

생년월일시 입력

양력/음력 선택 후 태어난 년·월·일·시를 입력합니다. 시간을 모르면 "시간 모름" 체크 가능.

2

성별 선택 + 진태양시 옵션

남성/여성 선택 후, 원하시면 진태양시 보정을 켜서 더 정밀한 분석을 받을 수 있습니다.
